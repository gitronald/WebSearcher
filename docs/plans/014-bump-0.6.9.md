# Tidy up and bump to 0.6.9

## Context

The uv migration (poetry-to-uv) is complete on the current branch. Before bumping to 0.6.9, we should fix latent bugs, wire up the unused captcha detection, drop the unnecessary pandas core dependency, and clean up migration artifacts. These are all small, focused changes.

## Commits

### 1. Fix bugs in component parsers

**`WebSearcher/component_parsers/general.py:106-108`**
- Line 107: `sub.attrs['class'] == 'd4rhi'` compares a list (BS4 always returns `class` as a list) to a string — always `False`. Fix: `'d4rhi' in sub.attrs.get('class', [])`
- Line 108: `parsed['sub_type'] == 'subresult'` is comparison, not assignment. Fix: `=`

**`WebSearcher/component_parsers/view_more_news.py:30`**

### 3. Add captcha detection to feature extractor

`has_captcha(soup)` exists in `webutils.py:86-88` but is never called.
